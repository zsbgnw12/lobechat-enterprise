## 反馈问题

发现 Bug 或有功能建议？在 GitHub 提交 Issue，我们会在此跟进。

- [**GitHub Issues**](https://github.com/lobehub/lobehub/issues/new/choose) — Bug 反馈、功能建议与讨论

提交前请先搜索已有 Issue，避免重复。Bug 请附上复现步骤；功能建议请说明使用场景。
