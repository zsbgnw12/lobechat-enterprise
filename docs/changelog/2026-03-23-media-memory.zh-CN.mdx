# 图片与视频生成重设计

这次更新聚焦于优化视觉创作流程，让图片与视频生成之间的切换更自然、操作更高效。

## 更新内容

这次首先重做了生成流程。图片与视频创作不再像两条割裂的路径，新界面让你在两种模式之间切换更直接。

记忆管理也变得更干净利落。需要重置上下文时，你现在可以一键清空全部记忆条目。

此外，Agent 内部架构也做了重构，重点是提升稳定性，并为后续能力扩展留出空间。

## 体验优化

- 修复压缩视图显示异常。
- 优化移动端菜单交互。
- 修正消息计数显示准确性。
